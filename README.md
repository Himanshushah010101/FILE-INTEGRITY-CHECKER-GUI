# File Integrity Checker GUI 🔐

A GUI-based cybersecurity tool to verify file integrity using SHA-256 hashing.

## 🚀 Features
- User-friendly interface
- File selection via GUI
- SHA-256 hash generation
- Detect file tampering

## 🖥️ Tech Stack
- Python
- Tkinter

## 📦 Installation

```bash
git clone https://github.com/yourusername/file-integrity-checker-gui.git
cd file-integrity-checker-gui
pip install -r requirements.txt
python app.py

## 📦 Requirements

**CMD**
- pip install watchdog
